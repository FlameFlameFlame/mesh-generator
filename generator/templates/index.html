Usage Guide

      
    

    

      

        
    
          
  1. Select or create a project in the `Projects` section, then click `Open Project`.
    2. 
          
  2. Add sites in `Site Management` using `+ Add Site`, then click on the map to place each site.
    3. 
          
  3. For each site, set priority and height inline; use the city checkbox to fetch city boundaries.
    4. 
          
  4. Optional: draw a bounding box in `Preparation` if you want to limit data download area.
    5. 
          
  5. Click `Download Data` to fetch roads, elevation, and prepare the planning grid.
    6. 
          
  6. Click `Filter P2P` to build candidate routes between sites.
    7. 
          
  7. Review and adjust parameters in `Optimization`, then click `Run Optimization`.
    8. 
          
  8. Use `Calculation Runs`, `Report`, and `Optimization Log` to inspect results and reload past runs.
    9. 
          
  9. Open `Layers` to control map overlays for preparation data and optimization outputs.
    10. 
          
  10. Open `Coverage` to run runtime tower coverage checks and inspect coverage metrics.
    11. 
          
  11. Click `Save` in `Projects` to persist current state and outputs to the selected project.
    12.
